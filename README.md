# Emfrp

Pure Functional Programming with your microcomputer.

# Requirement

* Ruby2.0 or later (and it's Gem client)

## Installation
You can get executables via RubyGems.
```sh
$ gem install emfrp
```

# Command-line-interpreter (REPL)
```sh
$ emfrpi
```

# Compiler
```sh
$ emfrp <src-file>
```
